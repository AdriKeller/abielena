Hallo und herzlich Willkommen auf der Website zum Spiel Barbasauteur

Ich habe dieses Spiel im Rahmen eines Schulprojekts programmiert: Es handelt sich um mein Informatik-Abi am Deutsch-Französischen Gymnasium Freiburg. Auf dieser Seite, die ebenfalls Teil des Projekts ist, erkläre ich Idee und Prinzip des Spiels, wie es funktioniert und auf welche Probleme und Schwierigkeiten ich bei der Programmierung/Realisierung gestoßen bin.



Referenzen

  • Das Programm wurde mit Python 2.7 geschrieben
  • Seine graphic interface wurde mit der library pygame gemacht
  • Um die Basics von pygame zu lernen, habe ich mit techwithtim.net gearbeitet
  • Um Probleme zu lösen, habe ich stackoverflow.com benutzt
  • Beim arbeiten mit pygame habe ich die Pygame Dokumentation benutzt
  • Um den Code zu verifizieren, habe ich die w3c Validation für HTML und CSS benutzt
  • Und zum Schluss ein sehr großes Dankeschön an Adrian Keller für seine Hilfe wenn ich Fragen oder Bugs hatte und stackoverflow.com mir nicht helfen konnte.

Elena Baruc-Westhäuser, TSBC2
